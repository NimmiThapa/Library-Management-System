# Library Management System

This is a Java-based Library Management System developed using NetBeans IDE.  
It manages books, students, and transactions such as issuing and returning books.

## 🔧 Technologies Used:
- Java (Swing for GUI)
- NetBeans IDE
- MySQL Database

## 📌 Features:
- Add, update, delete books and students
- Issue and return books
- View transaction history

## 🚀 How to Run:
1. Open the project in NetBeans.
2. Import `database.sql` into your MySQL server.
3. Configure DB connection in your code (if required).
4. Run the project.
