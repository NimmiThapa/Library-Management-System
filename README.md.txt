# 📚 Library Management System

This is a Java-based Library Management System developed using NetBeans IDE. It allows users to manage a collection of books, issue and return books, and track library records efficiently.

## 🚀 Features

- 📖 Add, update, delete books
- 👥 Manage borrowers and users
- 📅 Issue and return books
- 🔍 Search functionality for books and users
- 📊 Track issued/returned books and overdue records
- 🛡️ Basic login authentication system


## 🛠️ Technologies Used

- Java
- NetBeans IDE
- MySQL (for database)
- JDBC (Java Database Connectivity)

## 💻 How to Run

1. Open the project in NetBeans.
2. Make sure MySQL server is running.
3. Import the database (provided SQL file) into MySQL.
4. Update database credentials in the code if necessary.
5. Build and run the project from NetBeans.
